# Calendar 

Fork of [Liam Cain’s **Calendar**](https://github.com/liamcain/obsidian-calendar-plugin) plugin for Obsidian.

![Obsidian_NGf9Of9P0m](https://github.com/user-attachments/assets/2a644979-e9d7-40d7-983a-504801d79875)

## What’s different:
- Adds a **List view**  
  <img width="530" height="916" alt="image" src="https://github.com/user-attachments/assets/46ff7a05-ddbc-40bb-9ebf-cea3e4357dac" />  
- Group Daily Notes in 6 different options:  
  ![Obsidian_gCFGsavFxt](https://github.com/user-attachments/assets/6cf213d3-fd16-412a-82ee-11e7ccc31c4e) <img width="403" height="1068" alt="image" src="https://github.com/user-attachments/assets/c8bdbea8-7945-4da2-9e23-e169db0069a5" />  
  <img width="2441" height="1091" alt="image" src="https://github.com/user-attachments/assets/2fbc06bc-a492-4667-8966-3ce20a006d29" />
- Ability to add custom title to Daily Note:  
  ![Obsidian_6d5WIMLpth](https://github.com/user-attachments/assets/fc72e3ed-f374-4bfa-87ad-4c64ca9693f4)  
- Option to add custom color tags to Daily Notes via right-click  
  ![Obsidian_aTxtDpIzkf](https://github.com/user-attachments/assets/0116a570-9acf-4e6a-8390-5d4e463683c3)  
- Adds an optional title generation via `gemma3:4b`
  - When enabled, each daily note row in List view shows a small refresh icon to generate/update the title.
  - Uses your local Ollama server (default: `http://127.0.0.1:11434`)
  - Generated titles are stored in plugin data (it **does not rename files**)
